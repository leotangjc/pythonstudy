## requests模块
导入模块  

    import requests  
使用get方法获取网页    

    requests.get('https://....', data = {'kdy':'value'})  

或者使用外部引用的方式  

    KEY = {'kdy':'value'}
    requests.get('https://....', data = KEY)

发出请求后，requests会根据http头部的信息，判断出响应的编码。    
使用.encoding可以查看，或重新指定编码。  
request中有内置的json解码器，使用.json处理json数据  
    如：  
    
    r = requests.get('https://api.github.com/events')
    r.json()
